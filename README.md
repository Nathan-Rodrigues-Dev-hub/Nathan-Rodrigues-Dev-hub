## 🚀 Conceitos que aplico na prática

| Conceito | O que aprendi |
|---|---|
| 🏛️ **Clean Architecture** | Isolar a lógica de negócio de frameworks e infraestrutura |
| 🎯 **DDD** | Modelar o código para que ele reflita a regra de negócio |
| ⚡ **CQRS + MediatR** | Separar responsabilidades de leitura e escrita |
| 🧪 **Testes Unitários (xUnit)** | Garantir confiabilidade em cada funcionalidade |
| 🔗 **Microserviços** | Construir sistemas distribuídos e desacoplados |
| 🐇 **RabbitMQ + MassTransit** | Comunicação assíncrona entre serviços |
| 🔐 **JWT** | Autenticação segura implementada do zero |
| 🌐 **API Gateway (Ocelot)** | Centralizar e gerenciar rotas entre microserviços |
